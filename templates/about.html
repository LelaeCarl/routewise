About RouteWise

RouteWise is a decision support system for cross-border logistics between China and Kenya. It models multi-modal transport corridors as a weighted graph and uses shortest-path optimization to recommend routes that balance cost, transit time, and operational simplicity.

What it solves

Cross-border trade between China and Kenya involves sea, air, rail, and road legs that interact in non-obvious ways. A cheaper route might take weeks longer; a faster route might cost multiples more. RouteWise makes these trade-offs visible and structured so operators can make informed decisions rather than relying on intuition or spreadsheet estimates.

Multi-modal complexity

Sea, air, rail, and road handoffs across ports, airports, and ICDs.

Cost vs time trade-offs

Different routing objectives produce meaningfully different route outcomes.

Non-intuitive routing

Operators need structured recommendations, not raw graph outputs.

Decision transparency

Each route includes a rationale, leg-by-leg breakdown, and objective comparison.

Capabilities

Route planning — configure direction, origin, destination, cargo weight, and routing objective.

Multi-modal routing — compute optimal paths across sea, air, rail, and road segments.

Objective comparison — view results for lowest cost, fastest delivery, and balanced trade-off side by side.

Structured results — KPIs, route rationale, leg-by-leg table, and route path visualization.

Technical stack

Backend Python + Flask
Routing Dijkstra on weighted graph
UI Jinja + Tailwind CSS
Client Vanilla JavaScript
Data Seeded JSON network

Routing approach

The logistics network is modeled as a directed graph where nodes represent ports, airports, ICDs, and hubs. Edges carry cost and time attributes. Dijkstra's algorithm computes the shortest path using a weighting function determined by the chosen objective.

Academic context

RouteWise is designed as a graduation project. The interface exposes routing assumptions and trade-offs so evaluators can validate the algorithm's behavior and assess the quality of route recommendations.
