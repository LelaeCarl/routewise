About RouteWise

A decision support tool for intelligent cross-border logistics between China and Kenya.

Academic-friendly by design

Clear structure for later algorithm explanation.

Purpose

RouteWise helps users translate complex logistics trade-offs into clear route recommendations. Users can express a preference (cheapest, fastest, or balanced) and receive an easy-to-read summary.

Problem statement

Cross-border shipping is difficult because there are multiple transport modes, cost/time variability, and non-intuitive handoffs between ports, airports, ICDs, and inland delivery points.

System features (planned)

Multi-modal routing

Sea, air, rail, and road support.

Preference control

Cost vs time trade-off.

Two-way logistics

China ↔ Kenya routing.

Comparison view

Multiple options and trade-offs.

Tech stack

  • Backend Python + Flask
  • Templates Jinja
  • Frontend JavaScript + Tailwind CDN
  • Routing logic Phase 3+ (graph model)

Academic context

RouteWise is structured to support clear explanations of graph modeling and shortest-path methods. In later phases, the app will expose the assumptions behind cost/time calculations to help researchers and students validate results.
