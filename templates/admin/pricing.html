Current model

RouteWise currently derives per-leg shipment cost from a weight-based model implemented in the routing engine: cost = max(base_cost + weight_kg × cost_per_kg, minimum_charge).

For this phase, pricing is inspected via the network edge attributes (base cost, per-kg rate, minimum charge) used during computation. Editing is intentionally disabled to keep the system stable.
