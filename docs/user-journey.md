# User Journey – RouteWise

## 1. Overview

The system guides users through a simple flow:
1. Discover
2. Input
3. Compute
4. Understand

---

## 2. Homepage

User sees:
- Product explanation
- Quick route planner

Action:
- Click "Plan Route"

---

## 3. Route Planner

User inputs:
- Origin
- Destination
- Weight
- Preference

System validates input.

---

## 4. Processing

User clicks calculate.

System:
- Loads graph
- Runs algorithm
- Computes route

User sees loading state.

---

## 5. Results

User sees:
- Best route
- Total cost
- Total time
- Route breakdown

---

## 6. Exploration

User can:
- Compare routes
- Adjust inputs
- Recalculate

---

## 7. Knowledge

User explores hubs and logistics information.
