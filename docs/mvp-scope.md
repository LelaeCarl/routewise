# MVP Scope – RouteWise

## 1. Goal

Define what MUST be built for the system to be considered complete.

---

## 2. Included Features

### Core System
- Route calculation engine (Dijkstra)
- Graph-based network model
- Cost and time attributes

---

### User Interface
- Homepage
- Route planner page
- Results page
- Hubs page
- About page

---

### Input
- Origin
- Destination
- Weight
- Preference (cost vs time)

---

### Output
- Route path
- Total cost
- Total time
- Transport modes

---

### Logic
- Two-way routing
- Cost vs time weighting

---

### Data
- JSON dataset of nodes and edges

---

## 3. Optional Features (If Time Allows)

- A* algorithm
- Route comparison view
- Export results
- Improved visualization

---

## 4. Excluded Features

- Login system
- Real-time APIs
- Payment integration
- Full admin dashboard

---

## 5. Success Criteria

The MVP is complete if:
- Users can calculate routes
- Results are correct and clear
- UI is clean and usable
